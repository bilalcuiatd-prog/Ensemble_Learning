# Code Availability Statement

The source code used for molecular curation, descriptor generation, scaffold-stratified nested cross-validation, ensemble model training, calibration, applicability-domain analysis, and external validation is available in this repository. The pipeline was implemented in Python using RDKit, scikit-learn, LightGBM, NumPy, pandas, and matplotlib.

Private or third-party datasets are not included in the repository. Users should place the required internal and external CSV/TSV files in `data/raw/` and `data/external/` before running the scripts.
